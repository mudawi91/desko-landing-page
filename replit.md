# DeskGo Landing Page

## Overview
A static landing page for DeskGo - a simple, fast service desk built for small IT teams. The site includes a main landing page, login page, and dashboard page.

## Project Structure
- `index.html` - Main landing page with product information, features, and waitlist signup
- `login.html` - Login page
- `dashboard.html` - Dashboard page
- `logo.png` - DeskGo logo
- `favicon.png` - Site favicon
- `server.py` - Python HTTP server for serving static files

## Running the Project
The project uses a simple Python HTTP server to serve static files on port 5000.

```bash
python server.py
```

## Deployment
This is a static site that can be deployed using Replit's static deployment target, serving files from the project root directory.
